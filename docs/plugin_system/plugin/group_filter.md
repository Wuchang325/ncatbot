# GroupFilter 插件说明

GroupFilter 插件用于管理群组的白名单和黑名单，控制哪些群组可以使用机器人的功能。

## 功能
- **白名单管理**：允许指定的群组使用机器人的所有功能。
- **黑名单管理**：禁止指定的群组使用机器人的任何功能。
- **动态更新**：支持通过命令动态添加或移除群组的白名单和黑名单状态。

## 命令
- `gf_mode`：切换群组过滤模式。用法：`gf_mode [mode]`
  - 白名单模式：只有白名单中的群组才能使用机器人的功能。：
  - `white`：白名单模式
  - 黑名单模式：黑名单中的群组不能使用机器人的任何功能。：
  - `black`：黑名单模式
  - 无过滤模式：所有群组都可以使用机器人的功能。
  - `off`：无过滤模式
  - 使用 `gf_mode [mode]` 切换模式，例如 `gf_mode white` 切换到白名单模式。
- `gf_add`：添加群组到白名单或黑名单。用法：`gf_add [group_id]`
  - 例如：`gf_add 1234567890`
- `gf_del`：从白名单或黑名单中移除群组。用法：`gf_del [group_id]`
  - 例如：`gf_del 1234567890`
- `gf_list`：显示当前的白名单和黑名单。用法：`gf_list`
